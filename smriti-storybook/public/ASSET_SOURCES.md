# Asset Sources

## Audio (public/audio/)
Replace placeholder .mp3 files with real assets from:
- **ambient-loop.mp3**: Dark cinematic ambient (~2 min loop) — Pixabay "dark ambient" or Freesound.org
- **transition-whoosh.mp3**: Short whoosh SFX — Pixabay "whoosh"
- **quiz-correct.mp3**: Positive chime — Pixabay "correct answer"
- **quiz-wrong.mp3**: Soft buzz — Pixabay "wrong answer"
- **chapter-unlock.mp3**: Achievement unlock sound — Pixabay "unlock"
- **completion-fanfare.mp3**: Triumphant fanfare — Pixabay "fanfare"

All must be royalty-free / CC0.
